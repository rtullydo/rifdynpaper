Definition 3.5.

Suppose \(t \in \overline{\mathbb{D}}\) is a fixed point of a MÃ¶bius transformation \(\mathfrak{m}\) and define the multiplier \(\mathfrak{p}=\mathfrak{m}'(t) \text{.}\) We say that

  • \(t\) is attracting if \(0\lt |\mathfrak{p}|\lt 1\)
  • \(t\) is repelling if \(|\mathfrak{p}|>1\)
  • \(t\) is indifferent if \(|\mathfrak{p}|=1\text{.}\)

We further distinguish between rationally indifferent fixed points arising when \(\mathfrak{p}\) is a root of unity, and irrationally indifferent fixed points corresponding to \(\mathfrak{p}\) unimodular but not a root of unity.
