If \(\phi\) is a bidegree \((1,n)\) rational inner function, then for each fixed \(\lambda \in \mathbb{T}\text{,}\) the analytic function \(z_1\mapsto \phi(z_1,\lambda)\) is bounded, satisfies \(|\phi(z_1,\lambda)|=1\text{,}\) and is of degree \(0\) or \(1\text{.}\) Then either \(\phi(\cdot, \lambda)\) is a Möbius transformation for some values of \(\alpha(\lambda)\) and \(a(\lambda)\text{,}\) or \(\phi(\cdot, \lambda)\) is constant. The latter possibility corresponds to a collapsing fiber, and by Lemma 3.3, this occurs if only if \((\zeta_1,\lambda)\) is a singularity of \(\phi\) for some choice of \(\zeta_1\text{.}\)
